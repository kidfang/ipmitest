SAS3IRCU
========
This is a Userworld Tool (SAS3IRCU) for RAID configuration on LSI SAS3 Controllers that is designed to run from a host.
Phase 15 GCA Release Version 16.00.00.00
Release Date : 4-MAY-17
Package contains binaries for DOS, EFI x64/ARM, Windows x86/x64, Linux x86/x64/PPC64/ARM, FreeBSD i386/amd64, Solaris x86/SPARC, VMware.
Release Notes: IRCU_MPT_GEN3_Phase15.0-16.00.00.00.pdf
